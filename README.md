#  Hotel Billing System

A full-stack Hotel Billing System built using **Angular, Node.js, Express.js, and MySQL** that streamlines restaurant billing operations. The application allows administrators to manage categories and menu items, generate customer bills, and download professional PDF invoices with secure authentication.

## Features

- 🔐 JWT Authentication (Secure Login & Authorization)
- 📂 Category Management (CRUD Operations)
- 🍔 Menu Item Management (CRUD Operations)
- 🧾 Bill Generation with Automatic Total Calculation
- 📄 Download Bills as PDF using PDFKit
- 📜 Bill History Management
- 🗄️ MySQL Database Integration
- 🌐 RESTful API Architecture
- 📱 Responsive User Interface with Angular
